## Get Started

### Clone Project
  1. Open Cmd (Command Line)
  2. git clone https://github.com/khampheeraphopIT/Portfolio.git
  3. cd Portfolio
  4. npm install
  5. code .
  6. open terminal
  7. npm run dev
